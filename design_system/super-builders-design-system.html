06 / Motion

MOTION TOKENS

Hover the squares to feel each easing curve. Spring for rewards and badges. Ease-out for entrances. Bounce for CTAs. Fast for hover states.

--ease-spring · cubic-bezier(0.34, 1.56, 0.64, 1)
Badge unlocks, confetti, level-up animations. Overshoots then settles.
250ms
--ease-out · cubic-bezier(0.16, 1, 0.30, 1)
Page entrances, modals appearing, cards sliding in. Starts fast, decelerates.
400ms
--ease-in-out · cubic-bezier(0.87, 0, 0.13, 1)
Stage transitions, progress bar fills, smooth reveals.
600ms
--ease-bounce · cubic-bezier(0.68, -0.55, 0.27, 1.55)
CTA buttons, primary actions, call-to-register. Playful energy for teens.
300ms
--dur-fast · 150ms linear
Hover states, focus rings, color transitions. Instant feedback.
150ms
